1. Problem Statement

Margin volatility

  • Supply chain issues and volatile EUR/USD exchange rates threaten our 8% profit margin.
  • Goal: Predict extreme cost compressions proactively.
  • Data Source: fact_chromebook_margin_risk from Bloc 2 Warehouse.
Slide 2 / 8

2. The Machine Learning Model

Random Forest Classifier

  • Chosen for excellence in handling non-linear relationships (FX vs Shipping modes).
  • Features: EUR/USD, EUR/TWD, Delay Days, Freight Cost.
  • Label: Margin Impact Detected (0/1).
Slide 3 / 8

3. Serving Infrastructure

FastAPI Endpoint Integration

  • Model serialized using joblib resulting in a self-contained artifact.
  • FastAPI serves the artifact on a /predict-margin-risk REST endpoint.
  • Pydantic schemas ensure valid payload inputs.

5. CI/CD Operations

Continuous Integration via GitHub Actions

  • Every commit merges triggers mlops-ci.yml workflow.

6. Automated Retraining

Concept Drift Mitigation

  • System monitors macroeconomic drift on the EUR profile continuously.
  • If drift > 5% threshold detected, a Jenkins/Airflow trigger executes train_model.py on new warehouse data automatically.
